# FMS Shipment Tracking Chatbot

Production shipment-tracking chatbot using FMS and TMS APIs.

## Structure
- api/fms   → FMS production API calls
- api/tms   → TMS production API calls
- api/chat  → Chat intent + orchestration
- public    → Floating chat widget UI

## Deployment
Hosted on Vercel. Embedded into FMS by web team.
